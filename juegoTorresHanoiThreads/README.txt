Esta es la conclusion de un examen de datos, donde se realiza la implementacion de un juego
de la torre de Hanoi donde se usaron Templates, Nodo, Listas.

Cuando se corre la aplicacion esta va a mostrar:
( 1 )  Ingresar al Juego.
( 2 )  Mostrar el minimo numero de movimientos para ganar.
( 3 )  Mostrar el algoritmo completo paso a paso de la solucion.
( 0 )  Salir.

Cuando se ingresa a la opcion 1, este va a empezar el juego y se va a mostrar cada movimiento con valores.
Cuando se ingresa a la opcion 2, este va a mostrar cual es la cantidad minima de movimientos para ganar.
Cuando se ingresa a la opcion 3, este va a mostrar los pasos a realizar para completar el juego con los movimientos minimos.
